Neural Network Architecture

Legacy architecture figure retained for historical reference. It does not describe the current Conv1D plus stacked bidirectional GRU source model.

Architecture Highlights

  • Input 40 MFCC features per audio frame, reshaped to (1, 40) for GRU input
  • Branch 1 GRU(32) → LeakyReLU → GRU(128) → LeakyReLU
  • Branch 2 GRU(64) → LeakyReLU → GRU(128) → LeakyReLU
  • Residual Add Element-wise addition of both branches
  • Branch 3 & 4 Two further GRU branches (128→32, 64→32) with LeakyReLU
  • Residual Add Add branches 3, 4 and initial branch 1 output
  • Dense Head 1 Dense(64) → LeakyReLU → Dropout(0.2) → Dense(16)
  • Dense Head 2 Dense(32) → LeakyReLU → Dropout(0.2) → Dense(16)
  • Residual Add Add both Dense heads
  • Output Dense(16) → LeakyReLU → Dropout(0.5) → Flatten → Dense(6, softmax)
